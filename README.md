

 Learnify – Course Selling Web App

Learnify is a full-stack course selling platform designed to deliver a smooth and modern online learning experience. It allows users to explore courses, register, and interact with a clean and responsive interface while maintaining efficient backend services.


---

📌 Features

🔐 User Authentication (Sign up / Login)

📚 Browse and explore courses

🛒 Course selection and interaction

⚡ Fast and responsive UI

🔄 Backend API integration

🧠 Scalable architecture for real-world use



---

▶️ How to Run the Project

1. Clone the Repository

git clone https://github.com/your-username/learnify.git

2. Run Backend

cd backend
mvn spring-boot:run

Backend runs on:

http://localhost:8080

3. Run Frontend

Open frontend folder

Run using Live Server or open index.html
